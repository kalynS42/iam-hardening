# Identity & Access Management (IAM) Hardening
**Status:** In Progress

## Executive Summary
This repository documents IAM hardening work focused on improving access control (least privilege), documenting changes, and proving the result with verifiable evidence.

## What’s Inside
- `/proof` → screenshots/logs that confirm settings and results
- `/diagnostics` → issues + exact fixes
- `/configs` → exported settings or configuration notes (if used)
- `/scripts` → automation/scripts (if used)

## Verification Evidence
Proof artifacts are stored in `/proof` and are referenced in documentation as the source of truth.
